# Como Criar um Ambiente de Teste para Treinamento

Para que outras pessoas possam entender como funciona o sistema do OMIE, é necessário que essas pessoas tenham prática epossam entender os passos necessários para um bom gerenciamento. Mas como é possível fazer isso?<br/>
O maior problema é que o OMIE não possui um ambiente próprio para isso. Então é necessário criar de forma manual um ambiente controlado.<br/>
Para isso, crie clientes com CNPJ falsos, produtos que não existem, vendas falsas, compras falsas e ordens de serviço. Use como exemplos os dados abaixo: <br/>

- Produtos
    - Produto Final 1 (Teste)
    - Produto Final 2 (Teste)
    - Produto Final 3 (Teste)

- Cliente
    - Cliente SP (Teste)
    - Cliente RJ (Teste)
    - Cliente MG (Teste)

- Ordem de Serviço
    - Manutenção (Teste)
    - Aprimoramento (Teste)
    - Troca de Peça (Teste)

- Compra
    - Peças de Produto (Teste)
    - Placa de Hardware (Teste)
    - Peças 3D (Teste)

Com isso somente adicione 2 de 3 exemplos, para que a pessoa que estiver aprendendo possa entender como fazer todos os processos. E então começe criando processo como: compra, venda, ordem de serviço e movimentação de estoque.
