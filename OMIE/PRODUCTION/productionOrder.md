## **Ordem de Produção**

No link do vídeo acima, é apresentado como criar uma ***Ordem de Produção***, para que o estoque possa ser automaticamente atualizado. Para isso, é necessáro criar uma nova ordem. Se for um produto novo, é só clicar no ícone de clips para adicionar um novo. Salvando o produto a ser porduzido, você o adiciona a fabricação. Após adicionar todos os produtos que deseja na produção, agora é necessário por estimativa de término, MOB (Mão de Obra), MP (Matéria-Prima) e GGF (Gastos Gerais de Fabricação). Após passar por todas as etapas, quando chegar ao concluido é necessário informar quantos produtos foram produzidos, que local do estoque armazenar e data de conclusão. Uma coisa que pode ajudar muito é que é possível uma vinculação entre o pedido de venda e a ordem de produção. E, se a ordem de produção estiver vinculada a um pedido de venda com um vendedor, a cada etapa que o produto passa, o vendedor é notificado via email.
